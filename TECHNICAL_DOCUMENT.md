# 智能海报助手 (Smart Poster Generator) 技术文档

## 1. 项目概述
“智能海报助手”是一款专为社区零售和商超场景设计的全栈 Web 应用。该应用旨在帮助商家快速生成适合在微信群、朋友圈或线下张贴的商品海报，并利用 AI 技术生成亲切、实惠的营销文案。特别针对老年消费群体进行了视觉和交互上的深度优化。

## 2. 技术栈
- **前端框架**: React 19 (Vite 构建)
- **样式处理**: Tailwind CSS 4.0 (采用移动优先、响应式设计)
- **动画库**: Motion (用于标签切换和列表交互)
- **图标库**: Lucide React
- **海报渲染**: html2canvas (将 DOM 元素转换为高清图片)
- **AI 引擎**: Google Gemini 3.1 Pro (通过 `@google/genai` SDK 调用)
- **语言**: TypeScript (确保代码健壮性和类型安全)

## 3. 核心功能模块

### 3.1 批量海报生成器 (Poster Generator)
- **多图上传**: 支持一次性批量上传最多 20 张商品图片。
- **动态布局**: 
  - 系统根据上传商品数量（1-20个）自动计算网格布局（1-4列）。
  - 使用 `CSS Grid` 实现响应式排列。
- **实时预览**: 采用 React 状态驱动，用户修改商品名称或价格时，海报预览区域即时更新。
- **高清导出**: 利用 `html2canvas` 的 `scale: 2` 参数，导出 2 倍分辨率的 PNG 图片，确保打印清晰。

### 3.2 AI 文案助手 (AI Slogan Assistant)
- **智能生成**: 基于 Gemini 3.1 Pro 模型。
- **定向优化**: 针对老年群体进行了 Prompt Engineering（提示词工程）优化。
  - **语气**: 亲切、接地气，模拟邻里沟通。
  - **内容**: 自动强调“新鲜”、“实惠”、“健康”等核心卖点。
  - **视觉**: 自动加入 Emoji 表情，增加亲和力。
- **字数控制**: 自动控制在 30-50 字左右，适合社交媒体传播。

## 4. 适老化设计原则 (Elderly-Friendly Design)
本项目在视觉设计上严格遵循了适老化原则：
- **高对比度**: 采用红、黄、黑、白等高饱和度、高对比度的配色方案，确保视力不佳的用户也能清晰辨认。
- **超大字号**: 海报中的价格信息采用特大号粗体显示。
- **直观交互**: 
  - 减少层级，核心功能一目了然。
  - 增加“超市风”、“清晰大字”等长辈熟悉的视觉风格。
- **容错性**: 列表项支持随时删除和修改，操作逻辑简单直接。

## 5. 关键代码实现

### 5.1 海报样式切换逻辑
通过 `posterStyle` 状态控制预览区域的类名和渲染逻辑：
```tsx
const [posterStyle, setPosterStyle] = useState<'supermarket' | 'classic' | 'festive' | 'fresh'>('supermarket');
```

### 5.2 AI 提示词策略
在 `src/services/geminiService.ts` 中，通过精细化的提示词引导 AI 输出：
```ts
contents: `你是一个顶尖的社区营销专家...
要求：
1. 语气亲切、接地气...
2. 适当加入一些温馨的表情符号...
3. 字数控制在30-50字左右...`
```

### 5.3 性能优化
- **图片处理**: 使用 `FileReader.readAsDataURL` 进行本地预览，减少服务器压力。
- **滚动优化**: 为长列表添加了自定义样式的 `custom-scrollbar`，提升管理大量商品时的体验。

## 6. 部署与运行
- **开发环境**: `npm run dev` (运行在 3000 端口)
- **生产构建**: `npm run build` (生成静态资源至 `dist/` 目录)
- **环境变量**: 需要在 `.env` 中配置 `GEMINI_API_KEY`。
